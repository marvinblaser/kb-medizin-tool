# KB Medizin Tool - V1

Outil de gestion interne pour KB Medizin Technik (Biel-Benken, Suisse).

## Installation

```bash
npm install
npm run init
npm start
```

Ouvrez votre navigateur: http://localhost:3000

## Compte par défaut

- **Email:** admin@kbmedizin.ch
- **Mot de passe:** admin123

## Modules disponibles

- ✅ Connexion avec sessions
- ✅ Dashboard avec widgets personnalisables
- ✅ Tableau Clients (recherche, filtres, export CSV)
- ✅ Administration (utilisateurs, secteurs, équipements, logs)

## Technologies

- Node.js 20 LTS + Express
- SQLite3
- Vanilla JavaScript
- Leaflet + OpenStreetMap
